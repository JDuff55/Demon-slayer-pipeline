# Demon Slayer Pipeline

## Overview

The **Demon Slayer Pipeline** is a Java project with a fully automated **CI/CD pipeline** built using **Maven** and **GitHub Actions**.  
It includes unit testing, static analysis, and security checks to ensure code quality and secure deployments.

---

## Features

- Java-based application with utility classes (`NameValidator`) and test suite.  
- Unit testing using **JUnit 5**.  
- **Maven** for build automation and dependency management.  
- CI/CD pipeline via **GitHub Actions**: builds, tests, and static analysis run automatically on pull requests.  
- Static code analysis with **PMD** and **SpotBugs**.  
- Dependency security scanning using **OWASP Dependency-Check**.  
- Secure secret management: no credentials or keys are stored in the repository.
